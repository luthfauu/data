#Social Media Sentiment & Engagement Analysis

#Objective:
To analyze social media sentiment and user engagement patterns across Twitter, Instagram, and Facebook using cleaned Kaggle data and Power BI.

#Dataset:
Source: Kaggle (original link provided in dataset_link.txt)

#Tools Used:
Microsoft Excel – Data cleaning and preprocessing
Power Query – Data transformation
Power BI – Data modeling, DAX measures, and dashboard creation

#Data Preparation:
Removed null values and duplicates
Standardized sentiment labels
Created calculated columns and measures using DAX

#Key Insights:
Average engagement is 43 likes per post with 16,000+ total retweets.
Instagram shows the highest engagement in terms of users, posts, and average likes.
Positive sentiment dominates with “Positive”, “Joy”, and “Excitement” ranking highest.
Sentiments “Contentment” and “Excitement” receive the highest average likes.
User engagement peaks around 11 PM, indicating late-night activity trends.
Top trending hashtags include #Serenity, #Excitement, and #Gratitude.
Total posts show an upward trend across all platforms, reflecting growing adoption.
A strong linear relationship exists between likes and retweets, proving that highly liked posts attract more shares.
